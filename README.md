# 📦 Projeto Gest - E-commerce & Backoffice Integrado

Este repositório documenta o desenvolvimento da minha Prova de Aptidão Profissional (PAP), criada num contexto de estágio em ambiente empresarial. Este trabalho assinala o meu primeiro grande contacto com o planeamento de arquiteturas integradas de software, tendo sido totalmente desenvolvido na plataforma Low-Code **OutSystems**.

O principal foco foi ir além da construção de uma montra digital para o cliente (B2C) e resolver problemas reais de operações de negócio. O "Gest" consiste num ecossistema duplo: uma loja online de tecnologia e uma infraestrutura robusta de backoffice desenhada especificamente para a gestão de armazém.



## 🛠️ Tecnologias e Destaques Técnicos


* **Plataforma & Arquitetura:** OutSystems (Desenvolvimento Full-Stack Low-Code)
* **Gestão de Base de Dados:** Modelação e sincronização de dados entre a frente de loja e as operações internas.
* **E-commerce (Frontend):** Catálogo dinâmico de produtos de tecnologia e sistema completo de carrinho de compras.
* **Gestão de Stock (Backoffice):** Aplicação interna para controlo de inventário, atualizações de armazém e reflexão de stock em tempo real mediante as vendas.



## 🚀 Sobre a Execução do Projeto

Por ter sido arquitetado nativamente no ecossistema **OutSystems**, a execução ao vivo deste projeto depende da infraestrutura cloud e do Service Studio da plataforma. O código e a estrutura presentes documentam a modelação da base de dados, o design de ecrãs e a lógica de negócio (Actions) implementada durante o estágio.
